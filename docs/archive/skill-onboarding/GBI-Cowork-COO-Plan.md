# GBI Cowork COO — Comprehensive Operations Plan

## The Problem
You run 6+ content sites and a consulting portfolio across 2 ClickUp workspaces, with recurring tasks spanning SEO, content publishing, social media, outreach, site maintenance, product development, and admin — all on different cadences (weekly/biweekly/monthly/quarterly/bi-annual). The recurring work isn't systematically tracked, and you're relying on memory to trigger it. Meanwhile, 12+ tools (WordPress, Featured.com, Followr, RankMath, Linksy, AISEO, Lasso, Canva, Flodesk, SalesBlink, Looker, GA4) each need regular manual attention.

## The Solution
**ClickUp as the operations backbone** + **Cowork skills as the execution engine.**
ClickUp holds the "what" and "when." Skills encode the "how." When you open Cowork, the COO skill reads ClickUp, identifies what's due, and works through the queue with you.

## Content Publishing Checklist (Full — For `content-publisher` Skill)

Every article published on any GBI site follows this checklist:

1. SEO-optimized title with primary keyword
2. Featured image with descriptive alt text
3. RankMath setup: primary keyword, secondary keywords, meta description
4. Categories and tags assigned (using site-specific tagging scheme)
5. FAQ section with schema markup
6. YouTube video embed with source credit (when available)
7. Linksy interlinks to pillar/supporting content
8. Author assignment (guest author for bylined articles)
9. Lasso affiliate links where relevant
10. Preview QA (desktop + mobile)
11. **Post-publish**: 3-5 social posts created and scheduled in Followr staggered over 2-3 months
12. **Post-publish**: Newsletter inclusion (Flodesk for SafeMama, TBD for others)
13. **Post-publish (roundups only)**: Email each contributor with published link, Canva graphic, pre-written social post

## Limitations

- **Computer must be on**: Cowork scheduled tasks need the desktop app running
- **Approval required**: Every publish/accept/reject/send action requires your confirmation
- **ClickUp is source of truth**: Cowork reads from and writes back to ClickUp — doesn't replace it
- **No cross-session memory**: Can't read Claude.ai chat history — save key outputs to Google Docs
- **No video creation**: Can write scripts and storyboards, can't produce video files
- **No true multi-agent**: Cowork sessions are single-threaded — one conversation at a time, but skills let it switch contexts quickly
