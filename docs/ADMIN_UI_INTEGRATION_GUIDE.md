> into [`@pandotic/universal-cms`](../packages/cms-core/). The admin shell
> now lives at the `./components/admin` and `./components/admin/shell`
> subpath exports of that package.
>

> **[`docs/CONSUMER_ONBOARDING.md`](./CONSUMER_ONBOARDING.md)**.
>
> For the publisher / release workflow, see
> [`PUBLISHING.md`](../PUBLISHING.md) and [`docs/RELEASE.md`](./RELEASE.md).
