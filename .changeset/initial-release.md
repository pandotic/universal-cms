Initial release of Universal CMS — modular admin panel, data layer, and content management for Next.js + Supabase.

Features:
- 31-module config system with 4 presets (appMarketing, blog, directory, full)
- 17 data modules with SupabaseClient dependency injection
- Admin shell with sidebar, command palette, and AI chat
- 15 shadcn/ui components (bundled)
- Theme system with dark mode and server-side injection
- Auth middleware (requireAdmin, apiError)
- Security utilities (rate limiting, validation, CSP headers)
- AI chat integration (Anthropic Claude SDK)
